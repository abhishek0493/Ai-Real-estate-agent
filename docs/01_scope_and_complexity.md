# Complexity Level

Mid-scale SaaS product with production-grade architecture but limited dataset size.

# Must Have

- Multi-tenant architecture
- Secure tenant isolation
- Auth system (JWT)
- Admin dashboard for businesses
- Embeddable chatbot widget
- AI reasoning layer
- Property inventory database
- Lead tracking system
- Email integration
- Dockerized deployment
- CI/CD ready
- Unit + Integration tests

# Nice To Have

- CRM export (CSV)
- Conversation analytics
- Conversation memory
- Budget negotiation logic engine
- OpenAPI documentation

# Non-Goals

- Full-scale distributed microservices
- High-traffic scaling
- Real-time streaming architecture
